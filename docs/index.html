Jitsi-Helm Chart Repository

This page serves as the repository base for the Jitsi-Helm charts.

For Helm Users:

To use this repository, add it to your Helm client:

helm repo add jitsi https://jitsi-contrib.github.io/jitsi-helm/

For Documentation:

Please visit the main GitHub repository: Jitsi-Helm GitHub Repository

The chart catalog is available at index.yaml
